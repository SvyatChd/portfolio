КАК ОТКРЫТЬ САЙТ

1. Распакуйте ZIP-архив.
2. Внутри будет файл index.html и папка images.
3. Откройте index.html двойным кликом. Сайт откроется в браузере.

Важно: не вытаскивайте index.html отдельно без папки images, иначе фото не будут показываться.

Чтобы заменить фото:
- положите новые фото в папку images;
- назовите их photo-1.jpg, photo-2.png, photo-3.png, photo-4.jpg, photo-5.jpg;
- или поменяйте пути к фото прямо в index.html.
